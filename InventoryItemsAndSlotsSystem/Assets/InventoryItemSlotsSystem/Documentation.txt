In other projects you need to use only the following components:
  - InventoryComponent
  - InventoryConfiguration
  - InventoryManager
  - ItemComponent
If you want to change some some properties of the inventory and/ot items, or ui of the inventory then you can change some code in:
  - InventoryViewComponent
  - InventorySortingFunctions
  - ItemType.cs
  - ItemStack
The last one is InventorySlotComponent is purely technic.

To start using this system create an inventoryComponent gameObject, this will simulate your inventory, or items slot grid (like in Factorio), a list of badges or cards (like in Hogwarts legacy) etc. depending on the configuration you set in the inventoryConfiguration object. After you can add itemComponent to any gameObject which you want be able to be placed in the inventory. After create inventoryManager object, there should be just one inventoryManager for the whole game! After if you want to interact with inventory in the UI open this inventoryComponent through inventoryManager when you will change something in the inventory its view will be updated automatically. InventoryManager already provides some basic interaction with inventory UI: left mouse click and right mouse click, if you want you can add more interactions or change the existing one.
I recommend you to create prefabs of your different types of inventories and of all your items.

2. InventoryConfiguration stores information how inventory should behave and how it's UI should be drawn in the inventoryComponent



3. InventoryComponent is a component which represent the inventory itself, it has a list of slots which may or may not contain items. All slot positions in the inventory starts from (0,0) top-left corner. For your custom add function, the length of the sortedList should remain the same before and after your function! You can have your pass your custom sort functions and sort inventory however you wish!

5. InventoryViewComponent is component which visualizes provided inventoryComponent in the game UI by the rules described in the inventoryComponent's inventoryConfiguration. You can change how slot is selected at the SelectSlot method, but then you have to update DeselctSlot as well. You can change how to indicate rareness of the item in the IndicateItemRareness method. IMPORTANT this inventoryViewComponent has to be placed in the ScrollView UI gameObject!
